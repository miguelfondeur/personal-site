Hello

My name is Miguel Fondeur, I'm a UX/UI Engineer currently based in Charlotte, NC. What is a UX/UI Engineer you ask? Well, in short, I create the things you see and interact with on the web. And I make sure you feel good doing it. If you have a web project or just want to chat feel free to reach out.

More About Me:

I have a passion for building reusable web components and design language systems. I have extensive knowledge in building responsive web apps served across multiple devices and understand the intricacies and limitations of the mobile web. I'm constantly learning new skills that will help me build scalable, maintainable and performant front end code. Although I love using new technologies and javascript frameworks, my preference is to write clean, well commented native Javascript that works across all browsers.
